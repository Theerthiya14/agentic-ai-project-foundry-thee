# Agentic AI & Generative AI – Python Mini Projects

This repository contains a collection of **Python mini projects and automation programs** developed during a **4-Day Value Added Course on Agentic AI and Generative AI** conducted in our college by **Mr. Raju Varma**.

The projects focus on **Python programming, automation, APIs, and small real-world problem solving applications**.

---

## 📌 Projects Included

### 1. Job Alert System
Fetches job listings from **RemoteOK API** and sends notifications using **Twilio messaging**.

### 2. Message Automation
A Python script that sends automated messages using the **Twilio API**.

### 3. Password Strength Checker
Checks whether a password is **strong or weak** based on security rules.

### 4. Guess the Number Challenge
A simple interactive **Python game** where the user guesses a randomly generated number.

### 5. Match Scheduler (Tkinter GUI)
A **GUI-based match scheduling application** built using **Tkinter**.

### 6. QR Code Generator
Generates QR codes for different purposes.

### 7. GPay QR Code Generator
Creates a **QR code for Google Pay payments**.

### 8. Days Lived Calculator
Calculates the **number of days lived** based on the user’s birth date.

### 9. Digital Clock
A simple **digital clock application** using Python.

### 10. Bar Graph Visualization
Creates **bar graph visualizations** using Python.

### 11. Calendar Module Program
Demonstrates how to use Python’s **calendar module**.
